# Chrome Web Store Listing Draft

## Name

Element to PDF

## Short description

Export any HTML element as a high-quality PDF.

## Detailed description

Element to PDF saves the part of a web page you actually need. Instead of printing a full page, open the side panel, choose an HTML element, confirm the page highlight, and export that selected element and its child content as a PDF.

Use it to save element selections from receipts, application previews, forms, dashboards, cards, article sections, and other page areas where the full browser print output is too broad. It is designed for HTML to PDF workflows where you want to save, export, or capture one selected element instead of the entire page.

Core features:

- DevTools-style DOM tree in the Chrome side panel
- Search by tag, id, class, or text
- On-page element picker from the side panel
- Right-click "Pick element for PDF" workflow
- Element highlight before export
- Export of the selected element and descendant content
- DevTools Elements sidebar integration
- Frame support where Chrome extension access is allowed
- Local PDF generation and download through Chrome browser APIs

Privacy:

Element to PDF runs locally in your browser. It does not upload page content, generated PDFs, browsing history, or user activity to a remote server.

Chrome limitations:

Chrome extensions cannot run on Chrome internal pages, Chrome Web Store pages, most other extension pages, or restricted sandboxed frames. Local file pages require file access to be enabled by the user.

# Chrome Web Store Listing Draft

## Name

element-to-pdf

## Short description

Select any page element and export it to a PDF.

## Detailed description

element-to-pdf helps you turn a specific part of a web page into a PDF without printing the entire page.

Open the side panel, inspect the DOM tree, select an element, and export only that selected element. You can also start the picker from the right-click context menu and choose an element directly on the page.

Core features:

- DevTools-style DOM tree in the Chrome side panel
- On-page element picker from the extension panel or context menu
- Element highlighting before export
- Export of the selected element and its descendant content
- PDF saving through Chrome's built-in PDF generation flow
- Support for normal pages, popup windows, and frames where Chrome extension access is allowed

This extension runs locally in your browser. It does not upload page content or generated PDFs to a remote server.
